# Tech Hub Website Clone

A fully responsive modern tech company website clone built using HTML5, CSS3, and JavaScript.

This project demonstrates professional layout structuring, responsive design principles, reusable UI components, and clean frontend styling suitable for real-world web applications.

---

## 📌 Project Overview

This project replicates the structured layout style commonly used by leading global technology companies. The focus is on strong visual hierarchy, modern spacing, and responsive behavior across all screen sizes.

### Key Sections:
- Hero section with call-to-action
- Services section with reusable card layout
- Testimonials section
- Contact form UI with JavaScript interaction
- Sticky responsive navigation bar

---

## 🚀 Features

- Responsive navigation bar  
- Hero section with CTA button  
- Services cards with hover effects  
- Testimonials layout  
- Functional contact form (JavaScript interaction)  
- Sticky header behavior  
- Clean and consistent color system  
- Mobile-first responsive design  

---

## 🛠️ Technologies Used

- HTML5 – Semantic structure  
- CSS3 – Flexbox, styling, media queries  
- JavaScript (Vanilla JS) – Form submission handling  

---

## 📱 Responsive Design

The website adapts smoothly across:

- Desktop screens  
- Tablets  
- Mobile devices  

Flexbox and media queries ensure layout consistency and usability across screen sizes.
